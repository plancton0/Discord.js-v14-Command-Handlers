# Discord.JS V14 HANDLER COMMAND

Created by Speed!
## Used Packages And Environment

-   [DiscordJs >=v14](https://discord.js.org/#/)
-   [NodeJs >=20](https://nodejs.org/en/)

## Configuration File

Fill in the config.js

```
	token: "",
	owners: owners,
	color: "",
	clientId: ""
```

## Usage

To install dependencies

```bash
  npm i 
  npm install discord.js
```

Launch the bot

```bash
  node .
```
